BESTÄLL DITT PROGRAM

Fyll i formuläret för att beställa ett personligt anpassat program.

Träningsprogramet kommer att optimeras för att du ska göra framsteg i din träning. Målet med träningsprogrammet varierar beroende på vilket du väljer. Det kan vara större muskelmassa, ökad styrka, viktminskning eller en kombination. Du hittar en lista över program här.

Det är viktigt att du fyller i dina maxvikter så noggrant som möjligt. Vi använder den informationen för att optimera vikter, set och repititioner för att du ska göra framsteg så fort som möjligt.

I meddelande rutan kan du uppge om det är några övningar du vill undvika på grund av skador eller liknande.

  1. Fyll i namnet du vill ha på ditt formulär.
  2. Fyll i mail adressen som vi ska skicak ditt träningsprogram till.
  3. Fyll i vilket program som du vill få anpassat.
  4. Fyll i dina ungefärliga maxlyft i respektive övning.
  5. Fyll i övriga önskemål.
